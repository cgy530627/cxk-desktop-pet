# 🕺 蔡徐坤桌面宠物 (CXK Desktop Pet)

一个基于 Python 和 PyQt5 开发的趣味桌面宠物程序。蔡徐坤的虚拟形象会在你的桌面上自由移动、表演经典舞蹈动作，为你的桌面增添一丝乐趣和活力。

---

## ✨ 功能特性

- **流畅动画**：支持多种舞蹈和特殊动作，动画过渡自然。
- **智能移动**：宠物会在屏幕内自主移动，遇边反弹，模拟自然行为。
- **完全交互**：
  - **拖拽**：可用鼠标随意拖拽宠物到任何位置。
  - **点击**：点击宠物会有惊喜反应。
  - **菜单控制**：通过右键菜单或系统托盘图标，可随时切换特定动作、开始/停止。
- **轻量免安装**：提供打包好的单一可执行文件（`.exe`），下载后双击即可运行。

---

## 📦 下载与运行

### 直接使用（推荐）
对于只想体验的用户，请直接前往 [**Releases**](https://github.com/cgy530627/cxk-desktop-pet/releases) 页面下载最新版本的 `IKUN.exe`。

**运行要求**：
- 操作系统： Windows 10 或更高版本。
- 无需安装 Python 或任何其他依赖。

### 从源代码运行（开发者）
如果你想查看或修改代码：
1.  **克隆仓库**：
    ```bash
    git clone https://github.com/cgy530627/cxk-desktop-pet.git
    ```
2.  **安装依赖**：
    ```bash
    pip install PyQt5 pyinstaller
    ```
3.  **运行程序**：
    ```bash
    python main.py
    ```

---

## 🎮 使用指南

1.  **启动**：运行 `IKUN.exe`，宠物会出现在屏幕中央。
2.  **基础操作**：
    - **鼠标左键拖拽**：可自由移动宠物。
    - **鼠标左键单击宠物**：触发互动。
    - **鼠标右键单击宠物**：弹出控制菜单。
3.  **菜单说明**：
    - **捂裆派/华山劈**：执行指定的舞蹈动作。
    - **铁山靠/气波功**：执行指定的特殊动作。
    - **全屏移动**：让宠物随机选择舞蹈。
    - **停止移动**：宠物停止移动，进入待机状态。
    - **退出**：关闭程序。
4.  **系统托盘**：程序启动后，在桌面右下角的任务栏通知区域会有一个图标。右键点击它也可以进行控制，即使主窗口关闭，程序仍在后台运行。
